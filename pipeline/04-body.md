# Write Body

Read these files:
- `workspace/00-brief-validated.md` (product brief, content type, budgets)
- `workspace/01-research-youtube.md` (YouTube research)
- `workspace/02-research-x.md` (X/Twitter research)
- `workspace/03-hooks.md` (hook options from previous step)

You write the core content — everything between the hook and the CTA. Every line must earn its place. No filler. No fluff. Every word must justify its existence.

## Content Type Adaptations

### Launch Video / YouTube Ad
- Write the spoken script body
- Write for spoken delivery: short sentences, conversational, rhythmic
- Stay within word budget from the validated brief

### Twitter/X Thread
- Write tweets #2 through #N-1 (between hook and closer)
- Each tweet must standalone AND advance the thread narrative
- 280 chars max per tweet
- Use line breaks within tweets for readability
- Mix formats: insight tweets, proof tweets, "here's how" tweets, quote tweets

### Landing Page Hero (Long version only)
- Write 3 feature bullets (15 words each)
- Each bullet: [Benefit] — [specific proof or detail]
- Write 1 social proof line (15 words)

### Product Hunt Launch
- Write the description (260 chars max)
- Write the maker comment (300-500 words) — personal, honest, tells the story of why you built it

## Structure Options (for video/ad)

### The Problem-Solution Arc
1. Amplify the pain (use exact customer language from research)
2. Show why current solutions fail
3. Introduce the product as the inevitable answer
4. Prove it works (specifics, numbers, results)
5. Paint the after-state

### The Demo Flow
1. Quick context (the problem in 1 sentence)
2. "Let me show you" — walk through the product
3. Each feature tied to a specific pain point it kills
4. The "wow moment"
5. Stack the value

### The Story Arc
1. Start with a real scenario
2. The struggle (relatable, specific)
3. The discovery (the product)
4. The transformation (concrete results)
5. The implication (what this means for the viewer)

## Structure Options (for Twitter thread)

### The Revelation Thread
1. Context: why this matters right now
2. The insight/discovery (2-3 tweets)
3. Proof it works (data, screenshots, results)
4. How to do it (2-3 actionable tweets)

### The Listicle Thread
1. Items 1-N, each its own tweet, each with a concrete takeaway

### The Story Thread
1. The setup (1-2 tweets)
2. The turning point
3. The result/lesson
4. The bigger implication

## Writing Standards
- **Concrete > Abstract**: "saves 4 hours a week" beats "saves time"
- **Specific > General**: "replaces your 6-tab workflow" beats "simplifies your process"
- **Emotion > Logic**: Lead with how it feels, back with why it works
- **Rhythm**: Vary sentence length. Short punch. Then breathe.
- **Transitions**: No "furthermore." Use: "But here's the thing." / "Watch this."
